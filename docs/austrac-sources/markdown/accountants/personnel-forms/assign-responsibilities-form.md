# Assign responsibilities

This matrix is a summary of anti-money laundering and counter-terrorism financing (AML/CTF) obligations and which role will be responsible for meeting each obligation.

There are 4 kinds of roles that can be assigned these obligations:

- governing body (GB)
- AML/CTF compliance officer (CO)
- senior manager (SM)
- client -facing personnel and other personnel with an AML/CTF function (CP).
